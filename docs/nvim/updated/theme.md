# Theme

Color scheme plugin (`catppuccin/nvim`) providing a warm, pastel-based theme with high priority loading for consistent visual experience.

## Keymaps

*No custom keymaps - theme is applied automatically on startup.*

## Features

- Catppuccin color scheme with warm pastel colors
- High priority loading (1000) to ensure theme loads before other plugins
- Automatic colorscheme application on Neovim startup
- Consistent styling across all UI components and syntax highlighting
