# Code Suggestions

Autocompletion engine (`nvim-cmp`) with snippet support, LSP integration, and multiple completion sources.

## Keymaps

### Completion Control
| Keymap | Command | Description |
|--------|---------|-------------|
| `<C-Space>` | Trigger | Trigger completion manually |
| `<C-e>` | Abort | Abort completion |
| `<CR>` | Confirm | Accept selected completion |

### Documentation
| Keymap | Command | Description |
|--------|---------|-------------|
| `<C-b>` | Scroll up | Scroll documentation up |
| `<C-f>` | Scroll down | Scroll documentation down |

## Features

- Multiple completion sources (snippets, LSP, buffer, path)
- Bordered completion and documentation windows
- VSCode-style snippets via friendly-snippets
- Custom icons for different completion types
