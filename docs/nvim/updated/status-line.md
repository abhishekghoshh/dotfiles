# Status Line

Customizable status line plugin (`nvim-lualine/lualine.nvim`) with file icons and theme integration for enhanced editor information display.

## Keymaps

*No custom keymaps - status line displays information automatically.*

## Features

- Dracula theme integration for consistent visual styling
- File type icons via nvim-web-devicons integration
- Real-time display of file info, Git status, and LSP diagnostics
- Minimal configuration with default components and layout
