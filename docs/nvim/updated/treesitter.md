# Treesitter

Syntax highlighting and parsing plugin (`nvim-treesitter/nvim-treesitter`) with automatic language parser installation and enhanced code understanding.

## Keymaps

*No custom keymaps - syntax highlighting and indentation work automatically.*

## Supported Languages

### Systems Programming
- **C** - C language support
- **Rust** - Rust language support
- **Go** - Go language and go.mod files
- **Java** - Java language support
- **Scala** - Scala language support

### Web Development
- **JavaScript/TypeScript** - JS, TS, and TSX support
- **HTML** - HTML markup support
- **CSS** - CSS styling support

### Scripting & Data
- **Python** - Python language support
- **Lua** - Lua scripting support
- **SQL** - Database query support

### Documentation
- **Markdown** - Markdown and inline markdown support

## Features

- Enhanced syntax highlighting with language-specific parsers
- Automatic parser installation for supported languages
- Smart indentation based on syntax tree structure
- Language tree support for mixed-language files (e.g., JSX, Vue)
