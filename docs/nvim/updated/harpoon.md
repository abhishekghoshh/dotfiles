# Harpoon

File bookmarking and navigation plugin (`ThePrimeagen/harpoon`) for quick access to frequently used files.

## Keymaps

### File Management
| Keymap | Command | Description |
|--------|---------|-------------|
| `<leader>a` | `mark.add_file` | Add current file to harpoon |
| `<C-e>` | `ui.toggle_quick_menu` | Toggle harpoon quick menu |

### Navigation
| Keymap | Command | Description |
|--------|---------|-------------|
| `<leader>hn` | `ui.nav_next()` | Navigate to next harpooned file |
| `<leader>hp` | `ui.nav_prev()` | Navigate to previous harpooned file |

### Direct File Access
| Keymap | Command | Description |
|--------|---------|-------------|
| `<leader>h1` | `ui.nav_file(1)` | Jump to harpooned file 1 |
| `<leader>h2` | `ui.nav_file(2)` | Jump to harpooned file 2 |
| `<leader>h3` | `ui.nav_file(3)` | Jump to harpooned file 3 |
| `<leader>h4` | `ui.nav_file(4)` | Jump to harpooned file 4 |
| `<leader>h5` | `ui.nav_file(5)` | Jump to harpooned file 5 |
| `<leader>h6` | `ui.nav_file(6)` | Jump to harpooned file 6 |
| `<leader>h7` | `ui.nav_file(7)` | Jump to harpooned file 7 |
| `<leader>h8` | `ui.nav_file(8)` | Jump to harpooned file 8 |
| `<leader>h9` | `ui.nav_file(9)` | Jump to harpooned file 9 |

## Features

- Quick file bookmarking for frequently accessed files
- Sequential navigation through harpooned files
- Direct access to up to 9 bookmarked files
- Visual quick menu for file management
